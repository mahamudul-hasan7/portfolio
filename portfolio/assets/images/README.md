# Images Folder

Put your portfolio images here:
- `rakib.jpg` – Profile / hero image
- `rakib2.jpg`, `rakib3.jpg`, `rakib4.jpg` – Project/blog thumbnails
- `favicon.png` – Site icon

If images are in portfolio root, they will work. Move here for cleaner structure and update paths in `index.html` (e.g. `src="rakib.jpg"` → `src="assets/images/rakib.jpg"`).
